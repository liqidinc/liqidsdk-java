Liqid™ SDK Programming Reference for Java™ - Version 3.2.0

Enumerator ToggleState

Package:com.liqid.sdk

Description:

Describes a binary state of enablement/disablement

Values:

Enabled Indicates that the associated entity is enabled.
Disabled Indicates that the associated entity is disabled.
