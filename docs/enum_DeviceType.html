Enumerator DeviceType

Package:com.liqid.sdk

Description:

Represents a particular device type as a short string.
Note that redundant but differing values are not represented here.
This includes iba, fch, ramd, scm, vic, enc, nof, gof, pcpu, ncpu, gcpu

Values:

Enclosure Represents an enclosure
Ethernet Represents an ethernet NIC
Link Represents a general network link
Memory Represents configurable/switchable memory
Network Represents a general network link
Infiniband1 Represents an infiniband link
Infiniband2 Represents an infiniband link
SSD Represents a switchable SSD resource
Target Represents a target resource
PLX Represents a PLX chip resource
Compute Represents a CPU resource.
Returned from CPUDeviceInfo
CPU Returned from CPUDeviceInfo
GPU Represents a graphics processing unit resource
GCPU Represents a graphics processing unit resource
FiberChannel1 Represents a fiber channel link
FiberChannel2 Represents a fiber channel link
FPGA Represents an FPGA resource
